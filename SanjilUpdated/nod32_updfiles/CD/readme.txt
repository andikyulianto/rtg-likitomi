
             Create update CD 
             --------------------------------

Update CD for NOD32 antivirus system can be created
by copying(burning) the content of the subdirectory NOD_UPD of the current directory to the root (main)
directory of the CD. (Directory X:\NOD_UPD\ will be created, where X is the letter of your CDROM)

On a client workstation within the NOD32 Update module choose CDROM (X:\) from the server list and click
Update.
